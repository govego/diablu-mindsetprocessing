MindSet Processing

A library by Jorge C. S. Cardoso for the Processing programming environment.
Last update, 11/01/2014.

The MindSetProcessing library allows you to use the NeuroSky Mindset brainwave sensing headset with Processing.
It gives you access to the raw wave values, to the eSense meters (Attention and Meditation) and to the EEG values. It does not provide blink data. This library was implemented using the Think Gear Communications driver J2ME implementation provided in the Mindset Development Tools 2.1.

Download

Download MindSet Processing version 0.1.2 (2) in .zip format.

Installation

Unzip and put the extracted MindsetProcessing folder into the libraries folder of your Processing sketches. Reference and examples are included in the MindsetProcessing folder.
